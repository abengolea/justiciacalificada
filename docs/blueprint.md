# **App Name**: Qualified Justice

## Core Features:

- Registration/Login: Implement lawyer registration/login with professional license verification. Display confirmation on success or error.
- Courthouse Database: Allow browsing of courts sorted by province/jurisdiction, legal area, level, and city/locality. Each court displays its ratings. Includes search and filter functionalities.
- Rating System: Enable lawyers to anonymously rate courts in categories such as procedural efficiency and resolution time, and leave comments, including date of experience.
- AI-Powered Insight Generator: Use a generative AI tool to analyze lawyer comments, summarize key points and identify sentiment trends for each court.

## Style Guidelines:

- Primary color: HSL(210, 60%, 50%) - A professional blue that inspires confidence.
- Background color: HSL(210, 20%, 95%) - Light grey for a clean and neutral backdrop.
- Accent color: HSL(180, 60%, 40%) - A contrasting teal to highlight key actions and elements, like positive confirmations.
- The equivalent RGB hex values are: Primary: #1A75FF, Background: #F2F4F7, Accent: #269999
- Font pairing: 'Inter' (sans-serif) for body text, and 'Belleza' (sans-serif) for headlines to provide a professional and clean, but also artistic, readable appearance.
- Use minimalist icons to represent court characteristics (efficiency, resolution time) maintaining a clean and professional design.
- The layout will follow a 'desktop-first' approach with standard breakpoints to guarantee adaptability and a mobile-friendly experience.
